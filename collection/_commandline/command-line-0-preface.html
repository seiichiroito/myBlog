このチュートリアルはLearn Enough チュートリアルシリーズの一つ目です。このシリーズは、有名なRailsチュートリアルを含む9つのチュートリアルから構成さ れており、この Learn Enough Command Line to Be Dangerousから始まり、Learn Enough Action Cable to Be Dangerousという、Railsチュートリアルよりも応用的なチュートリアルで終 わります。

今回、より多くの人にこのチュートリアルのシリーズを届けるために翻訳をしましたが、是非と も元のサイトでチュートリアルを進めることもお勧めしたいです。

{% include component/figure.html num=0 %}

サイトは全て英語なので、最初始めるときには気が引けるかもしれませんが、プログラミングを学 ぶにあたって、英語を読まなければならない場面は遅かれ早かれやってくると思います。なので、 このサイトで英語に慣れるというのも一つの手です。また、サブスクリプションに登録すれば（ 通常$49/月ですが、翻訳時現在では$36.95/月で学べます）、動画でチュートリアルの内容を解 説しており、英語が分からなくても手順通りに進めることができ、加えて演習問題の答えも確認 することができます。ですのでこの翻訳版を読んだ後にでも是非覗いていただけると、こちらとし ても翻訳した甲斐があるというものです。

長々と前置きを書きましたが、最後に。一人でも多くの読者の方にこのチュートリアルを楽しんで いただけたら幸いです。
