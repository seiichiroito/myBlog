この章での大事なコマンドをテーブル1.2にてまとめてある。

コマンド 説明 使用例
echo <string> スクリーンに文字を出力する $ echo hello
man <command> コマンドのマニュアルを表示する $ man echo
^C トラブルから抜け出す $ tail ^C
^A 行の最初まで移動する
^D 行の最後まで移動する
^U 行の最初まで削除する
オプション-クリック クリックした場所にカーソルを移動させる
上下矢印 コマンド履歴の参照
cleanもしくは^L スクリーンを綺麗にする $ clear
exitもしくは^D ターミナルから出る $ exit

1.7.1 演習

1. クオテーションマークを含んだUse "man echo"と出力するコマンドを書きなさい。 Use man echoと出力されないように気をつけること。 ヒント: 文字列の中ではダブルクオートを使い、その文字列をシングルクオートで囲えばよい。

2. man sleepを実行し、ターミナルを5秒間寝させる(sleep)方法を見つけ、そのコマンドを実行しなさい。

3. 5000秒寝させるコマンドを実行し、その後それが1時間を超えることにあなたは気づいたので、Box1.4での方法を使用しこの問題から抜け出しなさい。
