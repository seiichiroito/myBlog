この章における大事なコマンドをテーブル2.1にまとめた。

コマンド 説明 使用例
> ファイルに結果を転送 $ echo foo > foo.txt
>> ファイルに出力を追加 $ echo bar >> foo.txt
cat <file> スクリーンに中身を出力 $ cat hello.txt
diff <f1> <f2> ファイル1、2の中身の比較 $ diff foo.txt bar.txt
ls ディレクトリ、ファイルをリスト表示 $ ls hello.txt
ls -l 長い形式でリスト表示 $ ls -l hello.txt
ls -rtl 長い形式で編集時間の遅い順 $ ls -rtl
ls -a 全てをリスト表示（隠しファイルを含む） $ ls -a
touch <file> 空のファイルを作成 $ touch foo
mv <old> <new> 古い名前から新しい名前に変更(移動) $ mv foo bar
cp <old> <new> 古いファイルを新しいファイルにコピー $ cp foo bar
rm <file> ファイルの削除 $ rm foo
rm -f <file> ファイルを即座に削除 $ rm -f bar

図2.5: シェイクスピアの一つ目のソネットのコピー＆ペーストしやすいバージョン（図1.11）

1. 図2.5のHTMLバージョンからテキストをコピー＆ペーストし、echoを使うことでシェイクスピアの一つ目のソネットの全て（オリジナル） のテキストを含むsonnet_1_complete.txtというファイルを作りなさい。 ヒント：echoを使う時、echo"のようにダブルクオートをセットで使わないと行き詰まることを思い出すかもしれないが（章1.3やBox1.4）、実はこれによって複数の行のテキストを入力することができようになる。 最後に閉じクオートを入れ、適切な名前のファイルにリダイレクトすることは覚えておくように。中身が正しいかはcatで確認できる。（図2.2）

2. fooという空のファイルを作成し、barという名前のファイルに変更し、それをbazにコピーする一連のコマンドを入力しなさい。

3. "b" から始まるファイルだけリスト表示するコマンドはなんだろうか？ヒント：ワイルドカードを使う。

4. rmを一度だけ使ってbarとbazの両方のファイルを削除しなさい。 ヒント：もし現在のディレクトリの中で"b"から始まるファイルがそれらしかない場合、前の演習のワイルドカードのバターンを使うことができる。
